Relatório de Projeto

Título: Desenvolvimento de 3 Projetos Interativos em Front End (Lâmpada, crônometro, calculadora)

Objetivos: Além de concluir com êxito os projetos, o objetivo é que os integrantes do grupo pratiquem conceitos de HTML, CSS e JavaScript. Desenvolvendo as habilidades gerais de cada um, voltado para o código e as respectivas linguagens citadas acima.

Introdução: Foi concebido durante a aula 9 variáveis de projetos, das quais os integrantes poderiam escolher apenas 3, uma de cada nível específico de dificuldade. No caso do nosso grupo, selecionamos desenvolver a lâmpada, o crônometro e a calculadora, aplicando certos quesitos necessários em cada um dos projetos.

Descrição das atividades: No tocante ao desenvolvimento da lâmpada, deveriamos trilhar pontos específicos para concluir o projeto com êxito. Ela deveria ter a funcionalidade de Ligar, Desligar e de Trocar a lâmpada quando ela quebrasse, no caso, após 4 vezes ligada, na quinta a mesma deve se quebrar.
Já no desenvolvimento do crônometro foi aplicado 3 botôes, um para iniciar, outro para pausar e o terceiro botão para zera-lo novamente. 
Por ultimo, foi desenvolvido uma calculadora funcional baseada na mesma contida no computador, desde as teclas e suas funções até os números e suas respectivas posições.
Todos os programas utilizaram linguagens como HTML, JavaScript e CSS!

Resultados: Após trabalharmos nos códigos para aplicar essas funcionalidades em cada um dos projetos, ficou evidente a aplicabilidade de cada um dos quesitos impostos para cada um dos elementos, desde a lâmpada até a calculadora, convergindo nos detalhes visuais dos programas desenvolvidos.

Discussão: As dificuldades que encontramos no desenvolvimento dos projetos foram diretamente relacionadas a sintaxe do javascript, visto que a parte lógica do programa é feita principalmente nele. Em sûma, o HTML e o CSS foram desenvolvidos de forma mais prática, por conta de atividades feitas diversas vezes em sala de aula.

Conclusão: Podemos concluir que, a finalidade da aplicação desses projetos foi atingida, além de desenvolvermos  diversas habilidades postas em prática durante a aula, conseguimos solucionar problemas e estrutura-los de forma mais compreensiva para o grupo. Com isso, fica evidente a proeminente capacitação dos integrantes do grupo com as atividades impostas, que não só desenvolveu nosssa capacidade de programar, mas também, de interação em grupos divergente aos que estamos acostumados.
